# etheremonQ

A simple command-line query tool for Etheremon accounts.

## Installation

```
git clone https://github.com/bkroeze/etheremonQ
npm install
#or
yarn install
```

## Usage

```
/path/to/etheremonQ/etheremonQ.js balance -a your-eth-account-address
```
